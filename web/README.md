# Webansicht

Die Datei `index.html` ist eine statische lokale Projektzentrale fuer das Sanierungsprojekt.

## Oeffnen

Im Browser diese Datei oeffnen:

```text
web/index.html
```

Die Links zeigen relativ auf Markdown-Dateien, PDFs und Bilder im Projektordner.

## Grundriss

Aktuell wird ein vorhandener Grundriss-Screenshot eingebettet. Eine saubere Neuzeichnung sollte erst erstellt werden, wenn die Masse eindeutig aus den Plaenen abgelesen oder vor Ort gemessen wurden.
