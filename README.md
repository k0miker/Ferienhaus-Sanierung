# Ferienhaus-Sanierung

Dieses Repository dokumentiert die Sanierung des Ferienhauses in Fuerstenau.

Ziel ist eine nachvollziehbare, lokale Projektakte mit:

- gesicherten Gebaeudedaten
- Originalunterlagen und Auswertungen
- offenen Fragen
- Entscheidungen
- Angeboten
- Kosten
- Bauablauf

## Arbeitsweise

- Originaldateien bleiben unveraendert in `99_quellen/` oder den passenden Quellenordnern.
- Auswertungen werden als Markdown-Dateien neben oder passend zu den Originalen abgelegt.
- Gesicherte Informationen werden in die zentralen Projektdateien uebernommen.
- Unsicherheiten und Rueckfragen werden sichtbar dokumentiert.
- Groessere Entscheidungen werden in `08_entscheidungen/` als ADR festgehalten.

## Zentrale Dateien

- `PROJEKTSTATUS.md`: kurze aktuelle Gesamtuebersicht
- `codex_memory.md`: Arbeitsgedaechtnis fuer Codex im lokalen Projekt
- `AGENTS.md`: dauerhafte Regeln fuer Codex
- `00_projekt/offene-fragen.md`: offene Punkte und Recherchebedarf
- `00_projekt/entscheidungslog.md`: chronologisches Entscheidungsprotokoll
- `09_bauablauf/backlog.md`: Aufgabenliste
- `web/index.html`: lokale Webansicht als Projektzentrale
